## ğŸ”§ Customization

### Changing Colors
Edit the CSS variables in `css/styles.css` to match your brand colors:

```css
:root {
  --primary: #2563eb;
  --primary-hover: #1d4ed8;
  --text-primary: #1f2937;
  --text-secondary: #4b5563;
  --bg-primary: #ffffff;
  --bg-secondary: #f9fafb;
  --border-color: #e5e7eb;
}

[data-theme="dark"] {
  --text-primary: #f9fafb;
  --text-secondary: #d1d5db;
  --bg-primary: #111827;
  --bg-secondary: #1f2937;
  --border-color: #374151;
}
```
